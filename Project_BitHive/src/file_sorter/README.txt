Сортувальник файлів застосовується для сортування різних типів файлів в певному каталозі. Спочатку застосунок перевіряє розширення файлу і, залежно від файлів поділяє їх по категоріям до свого каталогу, з відповідними назвами та кількістю.  
•	"Images" зображення ('JPEG', 'PNG', 'JPG', 'SVG');
•	"Video" відео файли ('AVI', 'MP4', 'MOV', 'MKV');
•	"Documents" документи ('DOC', 'DOCX', 'TXT', 'PDF', 'XLSX', 'PPTX');
•	"Audio" музика ('MP3', 'OGG', 'WAV', 'AMR');
•	"Archives" архіви ('ZIP', 'GZ', 'TAR');
•	"Відомі розширення": список розширень, які обробив сортувальник
•	"Невідомі розширення": список розширень, які не оброблялись
Крім того всі файли з кириличного алфавіту транслітеруються на латинський. 
Розширення файлів не змінюються після перейменування
Під час запуску програми в терміналі з’являється повідомлення:
«Вкажіть шлях до каталогу, який потрібно відсортувати». - Прописуємо шлях. Отримаєм список каталогів, з кількістю файлів в кожному.
Якщо шлях не існує, сортувальник просить ввести повторно шлях до каталогу, або завершити роботу з програмою за допомогою введення команди «exit». В терміналі повідомлення «Вихід з сортувальника».
