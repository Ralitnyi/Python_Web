Установка
Встановіть Python 3.x на свій комп'ютер, якщо він ще не встановлений.
Завантажте або склонуйте репозиторій з програмою на свій комп'ютер.

Запуск програми
запустивши програму, тобі запропонують ввести апіключ, який можна отримати на сайті https://platform.openai.com/account/api-keys. 
Якщо ключ введено неправильно, вас попросять ввести ключ по-новому. 
Далі потрібно просто написати цікаві для вас питання і бот з радістю відповість на них). 
Вихід здійснюється за допомогою нижче вказаних команд

Програма підтримує наступні команди:
exit, good bye, close - закрити бота 

Залежності
Програма використовує наступні залежності, які повинні бути встановлені:

openai - використання OpenAi API для запиту чата гпт

Встановіть залежності, виконавши команду pip install openai.
