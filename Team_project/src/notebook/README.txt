Нотатки

Ця програма дозволяє створювати, відображати, редагувати та видаляти нотатки. 
Вона працює з текстовими нотатками, які мають назву, текст та можуть мати теги для категоризації.

Встановлення

1. Встановіть Python 3.x на свій комп'ютер, якщо він ще не встановлений.
2. Завантажте або склонуйте репозиторій з програмою на свій комп'ютер.

Запуск програми
Відкрийте командний рядок або термінал.
Перейдіть до каталогу з програмою.
Активуйте віртуальне середовище, якщо ви використовуєте його.
Запустіть програму за допомогою команди python task_main.py.

Використання

Натисніть <Tab> для підсказки вводу команд
Програма підтримує наступні команди:

add "назва" "зміст нотатки" - Створити нову нотатку. Викликаємо: add "назва нотатки (не може містити пробілів)" "текс нотатки"
show "date|size|alp" - Виводить всі збережені нотатки. Можна сортувати за датою, розміром, алфавітом - show "date|size|alp". Текс нотаток виводиться до 300 символів
search "назва нотатки|тег" - Пошук нотатки за назвою або тегами. Писати назву можна частиною, тег треба писати повністю
close - Завершення роботи з нотатками та вихід в головне меню
help - Вивести список команд

Команди для роботи з нотаткою:
delete - Видалити обрану нотатку, закривається меню редагування нотатки
change "новий текс" - Змінити текст нотатки
tags "clean|add|change" - Робота з тегами. clean - видалити теги, add - додати нові теги (вводити нові теги через пробіл), change - замінити старі теги новими (вводити нові теги через пробіл) 
close - Закрити меню редагування нотатки
# Після виконання команд change, tags, меню редагування нотатки не закривається


Допомога
У разі потреби в додатковій допомозі, використовуйте команду help, яка виведе список доступних команд та їх пояснення.

Залежності
Програма використовує наступні залежності, які повинні бути встановлені:

prettytable: для красивого виводу даних у вигляді таблиці.
prompt-toolkit: для підсказок вводу команд 

Встановіть залежності, виконавши команду pip install -r requirements.txt.
